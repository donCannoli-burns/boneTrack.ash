#  <sup>:dagger:</sup><sub><ins>A Script to Better Manage :bone:Knucklebones\:bone: from :skull:["Skeleton of Crimbo Past"](https://wiki.kingdomofloathing.com/`Skeleton_of_Crimbo_Past):skull:</ins></sub><br/> 
 - Tracks, Records, and Displays, via CLI, the "Value Data" of the *Daily-Special* and *Knucklebone*.\
 - Logs All the Collected Data in KOL/data/boneTrack.`
## :small_blue_diamond: <sub><ins>_`boneTrack.ash` Tracks, Records, and Displays the Following:_</ins></sub>
| <ins>_`Daily`_</ins> | <ins>_`Value`_</ins> |
| :---                |      :---:      |
| the Skeleton of Crimbo Past | *Daily-Special Item* |
| *Daily-Special* | *Item* :hash: |
| Amount Of *Daily-Special* Owned | *Qty* |
| *Knucklebone* Store | :bone:  *Price* |
| Amount Of *Daily-Special* Purchased with :bone: | *Qty* |

| Daily *Knucklebone* Progression | :notebook: |
|      :---:      |      :---:      |
| :clipboard: | *Total* :bone: **in Invty**|
| :heavy_check_mark: | *Collected* :bone:|
| :x: | *Needed* :bone: | 
| *Daily-Special* | Tradeable true|\false | 
| --- | *Mall Price* :cut_of_meat: | 
| --- | :cut_of_meat: Per :bone: & "Value Per :bone:"* | 

| <ins>_Life-Time_</ins> | <ins>_*Daily-Special* Value_</ins> |
|      :---:      |      :---:      |
| Average *MpB* | :cut_of_meat: per :bone:      |
| Average *VpB* | Value per :bone:      |
| *Daily-Special* Purchased | *Qty*      |


>[!NOTE]
> "Average" meat per bone and value per bone is generated from
> the *Daily-Special* items **YOU** have logged with boneTrack.
> It will take a while to reflect a true *MPB/VPB* *number/value*.

## <sup>:dagger:</sup> <ins>To Install the Script, Use the Following Command in the KoLMafia CLI:</ins><br/>
      git checkout donCannoli-burns/boneTrack.ash
##### :small_blue_diamond: To Run the Script Just Type `boneTrack` into Mafia's CLI.
### :bell: <ins>You Will Receive a One-Time Prompt, to Enable or Disable the Wiki Auto-Launch.</ins><br/>
##### :small_blue_diamond: To [ disable | enable ] Wiki Auto-Launch Use:
       ##set boneTrackEnableWiki = [false|true]
#### :small_blue_diamond: To Reset Wiki Auto-Launch Prompt Use: 
       set boneTrackEnableWiki =" "

>[!NOTE]
> While script does not currently, combine legs after ascending, into a single data set.<br/>makings 
> It does log an additional data sheet, notated buy leg number.
