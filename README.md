# boneTrack.ash
A script for KOL Mafia. To track and log, the value Daily-Special of the Skeleton of Crimbo Past, and Knucklebones.

Tracks and logs the "Daily-Special" from the Skeleton of Crimbo Past. 
Tracks and Logs: item, item num, knucklebone price, is tradeable, mall price, if we purchased from SoCP.  
Tracks and Logs: "Daily-Specials" meat per bone & value per bone. && "Average" meat per bone and value per bone. 
  Note: "Average" meat per bone and value per bone is generated from the items you have logged with boneTrack.
  it will take a wihile to reflect a true number.
  
Logs all the collected data in KOL/data/bonetrack/ .

You will receive a one-time prompt, to enable or disable the Wiki auto-launch.
  To change setting use: set boneTrackEnableWiki=<false/true>  
  To reset use: set boneTrackEnableWiki=" "
